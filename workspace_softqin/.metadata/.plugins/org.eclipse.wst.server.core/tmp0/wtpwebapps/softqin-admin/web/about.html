关于软秦ABOUT

专注于科技地产、新兴产业项目投资

上海软秦投资管理有限公司，自2014年成立以来，专注于科技地产、科技园运营、新兴产业项目投资等领域，采用“轻+重”的资产运作模式：以重资产运作方式投资优质地产项目，获得优质物业；以轻资产运作方式进行资产运营，不断提升资产价值。目前已整合出龙软、磐界、磐邑、IM汇四个系列，努力构建“众创空间+孵化器+加速器+产业园”完整的科技孵化产业体系。总管理规模已达80万方,其中在建约30万方，分布于上海、杭州、南通、无锡、杭州、嘉兴、黄山等地。 未来，软秦投资将持续深耕所在领域，深度渗透全国各省城与重点中等城市，计划达到100家产业园连锁规模。欢迎各资源方以多种模式合作共赢，实现园区连锁化，成为影响中国产业发展新模式的领跑者。
